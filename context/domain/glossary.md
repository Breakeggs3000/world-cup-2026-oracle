# Domain glossary

Define terms agents and humans share. Prevents ambiguous instructions.

| Term | Definition |
|------|------------|
| **Agent** | An AI worker operating under a role from `agents/roles/` |
| **Crew** | A composed set of agents with a defined handoff process |
| **Task** | A scoped unit of work with acceptance criteria in `tasks/` |
| **Context** | Curated, durable project knowledge in `context/` |
| **Memory** | Agent-maintained facts in `memory/facts/` |
| **Scratch** | Ephemeral drafts in `workspace/scratch/` (not committed) |
| **Artifact** | Generated output in `workspace/artifacts/` (may be committed) |

_Add domain-specific terms as the product evolves._
